## Usage 
0. Press Ctrl+P and type `ext install vscode-lite-server` with a trailing space. 
0. Press Enter and restart VSCode.
0. Choose process from `Command Palette` or `Shortcut`.

### View SVG - `Alt+L L`
Launch lite-server

### Export PNG - `Alt+L S`
Stop lite-server
